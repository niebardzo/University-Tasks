def client.get_program ( )

Funckja pobierająca program - używając odczytu z zegara nazywamy plik do którego wpisany zostanie kod przesłanego programu, nastpenie łączymy się z serwerem i odebrane dane zapisujemy do utworzonego pliku.

Potem plik jest zamykany oraz nadawane mu są odpowiednie uprawnienia. Kolejnie program jest kompilowany i uruchomiony, a rezultat jest wypisany na ekranie.
